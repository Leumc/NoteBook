# 问题分析：

首先，本题指出，所有的船只到达时间都是递增的，所以在排除超时人员的时候，先到达的一定最可能先被排除，所以本题可以使用队列来实现。

另外，题目要求我们记录到当前船只到达时间**为止**24小时（86400s）内，到达的人员中不同的国籍数，所以应该开一个桶数组来记录各国籍的人数。

我们还需要每次都数出不同的国籍数，如果硬生生地数桶数组中不为0的项数的话，最坏的情况下需要循环$3\times 10^{10}$次，这个时间复杂度我们肯定是无法接受的。所以可以开一个变量，**实时追踪不同国籍数**，这样只需要在国籍数变化的时候更新这个变量即可。

==国籍数变化只会出现在以下两种情况下：==
1. **一种国籍的人数降为0，国籍数减一；**
2. **一种国籍的人到来之前，该国籍人数为0，国籍数加一；**
# 完整代码：

```cpp
#include <bits/stdc++.h>
using namespace std;
int n,ns[300005],cnt;
struct p {
	int time, nation;
};
queue<p> ps;
int main() {
	cin >> n;
	while (n--) {
		int t, k; cin >> t >> k;
		while (!ps.empty()&&ps.front().time <= t - 86400) {
			if (--ns[ps.front().nation] == 0)cnt--;
			//如果队列头部人员的国籍在减一之后等于0，那么国籍计数减一
			ps.pop();
		}
		while (k--) {
			int x; cin >> x;
			if (ns[x]++ == 0)cnt++;
			//如果到来的人员的国籍在此人之前等于0，那么国籍计数加一
			ps.push({ t,x });
		}
		cout << cnt << endl;
	}
}
```
