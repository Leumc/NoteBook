# 问题分析：

这道题要求询问我们输入数据中有多少不同的字符串，需要用到哈希算法。尽管并没有了解过，但实际上也许每个使用C++的人都用过它。

`unordered_map`和`unordered_set`是非常好用的数据结构，可以很方便地统计数据的出现次数，那么你是否想过，它们是怎么判断新加入的数据是否出现过呢？整型还好判断，那么字符串呢？总不能从头到尾比较一遍吧，时间复杂度奔$O(n^2)$了？这样的效率实在是太低了，此时就需要哈希算法出场了。

**哈希算法的核心是根据特定的算法为数据计算一个可以代表它的哈希值，并且保证两个不同数据的哈希值一定不同**。这样一来，只要我们判断一下两个数据的哈希值是否一致即可。
# 计算哈希值：

最简单哈希算法是**将字符串映射为一个P进制的整数，称为多项式滚动哈希**。类比二进制转十进制的公式，字符串$s$对应的P进制整数$H(s)$的十进制表达的公式就是：$$H(s)=(\sum^n_{i=0}s[i]\times P^{n-i-1})\bmod M$$
为了避免哈希值不会过大引起数据溢出，我们对其**取模M来约束数据范围**（基本上不会出问题的=ᗜωᗜ=）

我们首先选择P和M的值，需要保证：

* **P必须为大于字符集大小的素数**（这里取131）
* **M为一个大素数**（一般为`1e9+7`）

在这种情况下，不要使用`pow`函数，因为这个函数默认是用`double`类型进行计算的，整型截断以后会出现问题，而且每次都做一次完整的幂运算也非常耗时。

回看一下题目的数据范围，发现字符串的最大长度是1500，所以我们可以提前算出每个数位的基数，后面直接查表可以节省大量时间：

```cpp
long long powers={1};
for(int i=1;i<=1500;i++){
    powers.push_back((powers[i-1]*P)%M);
}
```

随后我们编写核心算法，计算哈希值：

```cpp
string s;cin>>s;
sz=s.size();
long long hash=0;
for(int j=0;j<sz;j++){
    hash=(hash+(s[j]-'a'+1)*powers[sz-j+1]+M)%M
    //s[j]-'a'缩小范围，+1去0
}
```

计算出哈希值后，我们可以自己写一个计数的逻辑，但实际上我们可以直接使用哈希集合。
# 完整代码：

```cpp
#include <bits/stdc++.h>
using namespace std;
unordered_set<long long> st;
vector<long long> powers={1};
const long long P=131,M=1e9+7;
int main(){
    int n;cin>>n;
    for(int i=1;i<=1500;i++){
        powers.push_back((powers[i-1]*P)%M);
    }
    for(int i=1;i<=n;i++){
        string s;cin>>s;
        int sz=s.size();
        long long hash=0;
        for(int j=0;j<sz;j++){
            hash=(hash+((s[j]-'a'+1)*powers[sz-j-1])+m)%m;
        }
        st.insert(hash);
    }
    cout<<st.size();
}
```
