# 问题分析：

题目说得很明白，相邻的国家两两进行比赛，得到一批圣者，然后继续相邻两两比赛，直到得到一个冠军，最终输出亚军是谁。并且由于国家数的2的幂，所以不会出现轮空的情况。

值得注意的是，能力值第二高的国家并不一定是亚军，如果在第一轮就遇到了能力值最高的国家，那么这个国家在很早就会被淘汰，从而无法进入到决赛，更无冠亚军只谈了。
另外，题目要求输出亚军的编号，这意味着我们还需要找个办法记录一下各个国家的编号。
# 分片比较法：

经典的递归思路，将所有国家分为左子片和右子片，一直分直到不能分为止：

```cpp
#include <bits/stdc++.h>
using namespace std;
int cts[130];//cts[i]为编号为i的国家的能力值
int beat(int l,int r){//beat将返回编号从l到r的范围内的胜者编号
	if(l==r)return l;//递归边界
	int mid=l+(r-l)/2;
	int ls=beat(l,mid),rs=beat(mid+1,r);//对左右子片分别递归
	return cts[ls]>cts[rs]?ls:rs//本层比较
}
int main(){
	int n;cin>>n;
	int m=1<<n;//1为2的零次幂，每左移一次加一次
	for(int i=1;i<=m;i++){
		cin>>cts[i];//读入数据
	}
	int mid=(1+m)/2;
	int ls=beat(1,mid),rs=beat(mid+1,m);
	//最后一层手动比较，便于确定亚军
	cout<<cts[ls]>cts[rs]?rs:ls;//写一个反的比较逻辑
}
```

如果我们定义树的父节点是两个子节点之间的较大者，那么就可以建立起这样一个完全二叉树来模拟比赛过程，此时，根节点就是冠军。而根节点的子节点中一定有一者是冠军，那么另外一个自然就是亚军。

首先我们需要定义节点的结构，我们需要储存国家的能力值（`power`）和编号（`id`），以及两个指向左右节点的指针`left`和`right`：

```cpp
struct node{
	int power,id;
	node* left;
	node* right;
	//构造函数
	node(int p,int i):power(p),id(i),left(nullptr),right(nullptr){}
}
```

随后我们需要依靠递归来把这个树给建起来（各国家的能力值按编号储存在`cts`数组中）：

```cpp
//build将会返回指向冠军节点的指针
node* build(int l,int r){
	if(l==r)return new node(cts[l],l);//递归边界
	int mid=l+(r-l)/2;
	node* leftnode=build(l,mid);//左片递归建树
	node* rightnode=build(mid+1,r);//右片递归建树
	node* root;
	if(leftnode->power>rightnode->power){//按照定义将root设为左右子节点中能力较大者
		root=leftnode;
	}
	else{
		root=rightnode;
	}
	return root;
}
```

建起树来之后我们只需要判断一下冠军节点是左节点还是右节点即可，另外一个一定是亚军：

```cpp
node* champion=build(1,m);
if(champion->id==champion->left->id){
	cout<<champion->right->id;//冠军来自左边，输出右侧节点id
}
else{
	cout<<champion->left->id;//反之输出左侧节点id
}
```
## 接下来附上完整代码：

```cpp
#include <bits/stdc++.h>
using namespace std;
int ps[130];
struct node {
	int power, id;
	node* left;
	node* right;
	node(int p, int i) :power(p), id(i), left(nullptr), right(nullptr) {};
};
node* build(int l, int r) {
	if (l == r)return new node(ps[l], l);
	int mid = l + (r - l) / 2;
	node* leftnode = build(l, mid);
	node* rightnode = build(mid+1, r);
	node* root;
	if (leftnode->power > rightnode->power) {
		root = new node(leftnode->power, leftnode->id);
	}
	else {
		root = new node(rightnode->power, rightnode->id);
	}
	root->left = leftnode;
	root->right = rightnode;
	return root;
}
int main() {
	int n;
	cin >> n;
	int m = 1 << n;
	for (int i = 1; i <= n; i++) {//输入数据
		cin >> ps[i];
	}
	node* champion=build(1, m);//建立完全二叉树
	if (champion->id == champion->left->id) {//寻找亚军
		cout << champion->right->id;
	}
	else {
		cout<<champion->left->id;
	}
}
```
