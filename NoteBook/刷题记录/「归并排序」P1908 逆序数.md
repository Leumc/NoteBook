# 题目分析：

本题要求求一个整数序列的逆序数，并且明确规定序列中的每一个数都$<=10^9$（也就是十亿），这意味着我们**可以用`int`数据类型来储存所有的数**。
### 回顾一下逆序数的概念：

这是一个来自于数学的概念，首先规定一个标准次序，如果两个数在序列中的位置不满足这个标准次序，那么这两个数就构成了一个逆序对，而逆序对的个数就是逆序数。

一般情况下，会选择从小到大排列为标准次序（本题也是如此），所以逆序对的定义就是：**靠前的一个数比靠后的一个数大（不包括等于）的一组数对**。例如对于序列$[0,2,1,5,7,3]$来说，$[2,1]$是一个逆序对，$[5,3]$也是一个逆序对。（所以基本上是**排列组合**的问题）
### 先暴力一点：

如果要求一个逆序数，那么实际上很简单，只需要**从首项开始遍历，每一次都将结果加上当前项之后比它小的数的个数即可**。那么时间复杂度怎么样呢？$O(n^2)$，慢死了。
### 什么造成了这样的结果？

计算机一次只能”看到“一个数据，所以每次都需要对当前项之后的更小项进行计数。那么，如果当前项之后的更小项的个数变成了理所应当的呢？比如，给它**排个序如何**？
### 神奇的巧合——归并排序：

归并排序的性质就是**在每一次合并操作时，左右两子数组都是保持有序状态**。如果进行的是升序排序，那么左右两个子数组都是升序的。换言之，**如果子数组的某一项比一个数大，那么就证明这一项之后的所有项都比这个数要大，一次的累加量就等于子数组剩余元素的数量。**
### 一石二鸟：

既然知道了归并排序的这种性质，那么就可以进行操作了。
首先，我们需要一个**标准的归并排序**：

```cpp
void merge(vector<int> &nums,int left,int mid,int right){
	vector<int> temp;
	int i=left,j=mid+1;
	while(i<=mid&&j<=right){
		if(nums[i]<=nums[j]){
			temp.push_back(nums[i++]);
		}
		else{
			temp.push_back(nums[j++]);
		}
	}
	while(i<=mid)temp.push_back(nums[i++]);
	while(j<=right)temp.push_back(nums[j++]);
	int n=right-left+1;
	for(int k=0;k<n;k++){
		nums[left+k]=temp[k];
	}
}
void mergeSort(vector<int> &nums,int left,int right){
	if(left>=right)return;
	int mid=left+(right-left)/2;
	mergeSort(nums,left,mid);
	mergeSort(nums,mid+1,right);
	merge(nums,left,mid,right);
}
```

我们发现，**在第9行的位置，出现了需要的条件（大于且不等于）**。创建一个静态整型变量来储存结果，**由于结果可能非常大，有可能爆`int`，所以使用`long long`保险**：

```cpp
static long long cnt=0;//置于merge函数之前
```

然后**在合并阶段统计逆序数**：

```cpp
cnt+=mid-i+1;//置于第9行else块中
```

**经此，在排序完成的同时，逆序数也就求出来了。**
归并排序的时间复杂度是$O(n\log n)$，空间复杂度是$O(n)$，相比于暴力求解，提升显著。

# 完整代码：

```cpp
#include <iostream>
#include <vector>

using namespace std;

long long cnt=0;

void merge(vector<int> &nums,int left,int mid,int right){
    vector<int> temp;
    int i=left,j=mid+1;
    while(i<=mid&&j<=right){
        if(nums[i]<=nums[j]){
            temp.push_back(nums[i++]);
        }
        else{
            temp.push_back(nums[j++]);
            cnt+=mid-i+1;
        }
    }
    while(i<=mid)temp.push_back(nums[i++]);
    while(j<=right)temp.push_back(nums[j++]);
    int n=right-left+1;
    for(int k=0;k<n;k++){
        nums[left+k]=temp[k];
    }
}

void mergeSort(vector<int> &nums,int left,int right){
    if(left>=right)return;
    int mid=left+(right-left)/2;
    mergeSort(nums,left,mid);
    mergeSort(nums,mid+1,right);
    merge(nums,left,mid,right);
}

int main(){
    int n;
    cin>>n;
    vector<int> nums;
    for(int i=0;i<n;i++){
        int t;
        cin>>t;
        nums.push_back(t);
    }
    mergeSort(nums,0,n-1);
    cout<<cnt;
}
```
