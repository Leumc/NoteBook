题目分析：

很明显，题目希望我们根据中序遍历和前序遍历的结果，重建出这棵二叉树，然后再进行后序遍历。所以，要解这道题，要先明白二叉树的三种遍历方式，即前序遍历、中序遍历和后序遍历，到底是什么。

要遍历一个二叉树，最常规的做法自然是使用深度优先搜索算法。我们需要从左右两个方向进行递归深入，这样一来，就有一个问题：我们应该在何时对节点进行操作？

二叉树的三种遍历过程

调整二叉树，改成从根节点向左右延长出两个长条，可以很好地观察出三种遍历的特点： - 前序遍历：先访问根节点，然后左右侧都正序遍历。 - 中序遍历：先左侧倒序遍历，再访问根节点，最后右侧正序遍历。 - 后序遍历：对左右侧都倒序遍历，最后再访问根节点。

回到这道题，由于中序遍历的访问在递归函数之间，所以被访问节点就天然地形成了一道分界线：在某个节点之前被访问的节点必然在树上处于该节点的左子树中，而在之后被访问的节点也必然在右子树中。

再看前序遍历，它的处理逻辑处于递归函数之前，这意味着在访问了一个节点以后，它的下一步就是对左右子树进行递归。 在这个时候，我们就可以从中序遍历序列中找出这个节点的左右子树部分 （不存在找不到的情况，稍微想想就知道了）。 紧接着程序进入左子树进行下一层递归，在这层递归里，我们依然可以进行左右子树的划分。以此类推，就可以重建出这棵二叉树。

#include <bits/stdc++.h>
using namespace std;
string in, pre;
int c=1;
int ls[100005], rs[100005];
int dfs(int l, int r) {
	if (l > r)return 0;
	int i = l;
	for (; i <= r; i++) {
		if (in[i] == pre[c]) {
			c++;
			flag = true;
			break;
		}
	}
	ls[i] = dfs(l, i - 1);
	rs[i] = dfs(i + 1, r);
	return i;
}
void post_order(int i) {
	if (ls[i]!=0)post_order(ls[i]);
	if (rs[i]!=0)post_order(rs[i]);
	cout << in[i];
}
int main() {
	cin >> in >> pre;
	int n = pre.size();
	in = " " + in; pre = " " + pre;
	post_order(dfs(1, n));
}
